# Smart Desktop Assistant

A simple desktop assistant built using Python and Tkinter.  
It helps users perform common computer tasks through a graphical interface.

## Features

- Open applications
- Execute basic system commands
- Simple and clean GUI
- Easy to extend with new commands

## Technologies Used

- Python
- Tkinter (GUI)

## Project Structure

smart-desktop-assistant-pro
│
main.py
gui.py

README.md

## How to Run

1. Clone the repository

git clone https://github.com/akhilakasukurthi/smart-desktop-assistant-pro.git

2. Go to project folder

cd smart-desktop-assistant-pro

3. Install requirements

pip install -r requirements.txt

4. Run the program

python main.py
